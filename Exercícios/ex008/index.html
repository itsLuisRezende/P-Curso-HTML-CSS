Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B. (não Semântica)

Nesta frase temos um termo em negrito usando a tag STRONG. (Semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I. ( não Semântica)

Nesta frase temos um termo em itálico usando a tag EM. (Semântica)

Texto marcado

Quando quisermos marcar um texto dessa maneira, utilizamos a tag MARK.

Texto Grande e Texto Pequeno

Quando quisermos que o texto fique grande desta maenira, utilizamos a tag BIG.

Quando quisermos que o texto fique pequeno, utilizamos a tag SMALL.

Texto Deletado

Quando houver a necessidade de deletar um texto dessa forma, utilizamos a tag DEL.

Texto Inserido

Para inserir um texto dessa forma, utilizamos a tag INS.

Também há outra maneira de inserir em um texto, utilizando a tag U. No entanto esta tag é não semântica, por tanto, nao recomendada.

Texto sobrescrito

Exemplo: utilize a equação x20 + 3. ( utilizamos a tag SUP para sobrescrever o "x elevado a 20").

Texto subscrito

Exemplo: Nosso corpo precisa de muito H2O. (para subscrever o numero 2 de "H2O", utilizamos a tag SUP).
